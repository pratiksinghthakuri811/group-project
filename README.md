#  About Project

##  Features

### 

###  Player Management



1)Add, update, delete and search players

2\) Store player details including jersey number, name, age, position, fitness, goals, injury and suspension status

3\) Duplicate jersey number detection





### &nbsp;Team Management



1. Register and configure teams with coach and staff details
2. Choose tactical formations (4-4-2, 4-3-3, 3-5-2, etc.)
3. Assign players to teams using jersey numbers
4. Remove players from squads
5. Delete teams and automatically release players as free agents
